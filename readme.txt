src/all_questions.php : Affiche l'ensemble des questions posées sur le site
src/connexion.php : Permet de se connecter 
src/editing.php : Permet d'éditer les questions et les réponses, au travers de 2 fonctions implentées dans ce fichier
src/forum.php : Permet de créer des questions, puis de les publier sur le site
src/home.php : Home page, affiche les questions les plus récentes et les plus populaires, ainsi que tout des liens vers les pages voulues par les consignes.  
src/inscription.php : Permet de s'inscrire au site 
src/menu.css : Gère le css du site.
src/menu.php : Fichier qui gère le menu, et qui démarre la session.
src/reponse.php : Page qui gère les réponses, permet de poser des réponses et de liker
src/SQL_table_connexion.php : Permet de se connecter à la table SQL
src/unanswered_questions.php : Permet d'afficher les questions posées qui n'ont pas encore de réponse
src/user_profile.php : Permet d'afficher le profil de l'utilisateur connecté, ses questions et ses réponses, et d'éditer tout cela
src/vote_function.php : Permet de voter/downvoter, avec vérification du nombre de vote par questions, du fait q'un utilisateur ne puisse pas voter pour ses réponses, et changement de vote/downvote possible.
